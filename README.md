# git-ghosts

Find deleted files, zombie branches, and orphan commits in git repositories.

```
$ git-ghosts scan .
$ git-ghosts report .

----------------------------------------
Category                  Count
----------------------------------------
Ghost Files               496
Zombie Branches           0
Orphan Commits            23
----------------------------------------
```

## Install

```bash
cargo install --path .
```

Or from source:

```bash
git clone https://github.com/genideas-labs/git-ghosts.git
cd git-ghosts
cargo build --release
# binary at ./target/release/git-ghosts
```

## Usage

### Scan a repository

```bash
# Scan current directory
git-ghosts scan

# Scan a specific repo
git-ghosts scan /path/to/repo

# Custom zombie threshold (days)
git-ghosts scan --threshold 7
```

### View cached report

```bash
git-ghosts report /path/to/repo
```

### Preview cleanup (dry-run only)

```bash
git-ghosts clean --dry-run /path/to/repo
```

## Daily scan with Telegram notification

`scripts/daily-scan.sh` scans all git repos in a directory and sends a summary to Telegram.

### Setup

```bash
# Set environment variables
export AW_TELEGRAM_BOT_TOKEN="your-bot-token"
export AW_TELEGRAM_CHAT_ID="your-chat-id"
export GIT_GHOSTS_SCAN_DIR="$HOME/projects"  # optional, defaults to ~/genideas-labs
```

### Run manually

```bash
./scripts/daily-scan.sh
```

### Cron (daily at 8am)

```bash
crontab -e
# Add:
0 8 * * * /path/to/git-ghosts/scripts/daily-scan.sh >> /tmp/git-ghosts.log 2>&1
```
